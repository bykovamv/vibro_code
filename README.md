# Виброкодинг — Учебный проект по веб-разработке

## 📖 Описание

Это учебный проект для изучения основ веб-разработки: **HTML**, **CSS** и **JavaScript**.

Здесь собраны простые примеры веб-страниц с подробными комментариями к коду. Каждый файл демонстрирует базовые концепции вёрстки и программирования в браузере.

## 📁 Структура проекта

| Файл | Описание |
|------|----------|
| `index.html` | Главная страница с кнопкой, меняющей цвет фона. Демонстрирует работу с событиями и DOM |
| `greeting.html` | Страница с персональным приветствием. Показывает использование `prompt` и условных конструкций |
| `hello.html` | Минималистичная страница «Привет, мир!». Базовый пример HTML-документа |

## 🎯 Чему можно научиться

- ✅ Структура HTML-документа
- ✅ Подключение CSS-стилей
- ✅ Работа с Flexbox для центрирования элементов
- ✅ Обработка событий JavaScript (клик, загрузка страницы)
- ✅ Манипуляции с DOM (изменение стилей, текста)
- ✅ Диалоговые окна браузера (`prompt`, `alert`)
- ✅ Генерация случайных чисел
- ✅ Адаптивная вёрстка (`viewport`, `rem`, `%`)

## 🚀 Быстрый старт

Просто откройте любой `.html` файл в браузере:

```bash
# Через проводник Windows — дважды кликните по файлу
# Или через командную строку:
start index.html
```

## 📝 Как использовать

1. **Изучайте код** — все файлы содержат подробные комментарии на русском языке
2. **Экспериментируйте** — меняйте цвета, размеры, текст
3. **Проверяйте результат** — обновляйте страницу в браузере

## 🔧 Требования

- Любой современный браузер (Chrome, Firefox, Edge, Safari)
- Текстовый редактор для изменений (VS Code, Notepad++, Блокнот)
